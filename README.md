# ActivityLifecycle

[![Kotlin](https://img.shields.io/badge/Kotlin-1.8+-blue.svg?logo=kotlin)](https://kotlinlang.org)
[![Platform](https://img.shields.io/badge/Platform-Android-green.svg?logo=android)](https://www.android.com/)

Тестовое приложение для изучения жизненного цикла Activity и LifecycleObserver в Android.

## О проекте

Приложение демонстрирует все состояния жизненного цикла Activity с визуальной обратной связью через Toast уведомления и логирование через кастомный LifecycleObserver.

**Основной функционал:**
- Отслеживание всех состояний жизненного цикла Activity
- Кастомный LifecycleObserver для логирования событий
- Toast уведомления при смене состояний
- Отображение текущего состояния Lifecycle

## Технологии

* Kotlin
* Activity Lifecycle
* LifecycleObserver
* LifecycleOwner
* Toast notifications
